# Django_Meetups_Project

A simple example of Django capabilities.

## Requirements

- Python 3.10
- Django 4.0

## Installation

To install and run this project, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/PartORG/Django_Meetups_Project.git
   cd Django_Meetups_Project
   ```

2. Create a virtual environment (optional but recommended):
   ```sh
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install dependencies:
   ```sh
   pip install Django==4.0
   ```

## Usage

To run the project, execute the following command:

```sh
python manage.py runserver
```

This will start a development server on `http://127.0.0.1:8000/`. You can access the application in your web browser to see it in action.
